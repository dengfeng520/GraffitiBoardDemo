### GraffitiBoardDemo
>画板demo,实现撤销，橡皮擦，保存到相册等功能

使用方法，
```
@property (strong, nonatomic) DrawView *drowview;

_drowview = [[DrawView alloc]initWithFrame:self.view.bounds];
    [self.view addSubview:_drowview];
```
